# Bienvenida a WN++

WN++ es un lenguaje de programación dinámico con sabor chileno. Si llegaste hasta acá, ya sabes lo que necesitas saber.

## ¿Qué puedes hacer en este tour?

Cada lección tiene una explicación a la izquierda y un editor a la derecha. Escribe código, presiona **▶ ejecutar**, y el resultado aparece abajo.

El código corre directamente en tu browser — no hay servidor, no hay instalación.

## Tu primer programa

El código de la derecha ya tiene un ejemplo. Ejecútalo.

```wn
lorea("¡Wena wn++!")
```

`lorea` es la función para imprimir en pantalla. El equivalente de `print`, `console.log`, o `println!` — pero con más sabor.
