En WN++ hay dos formas de guardar un valor: `wea` para cosas que pueden cambiar, y `duro` para cosas que no.

## `wea` - variable mutable

```wn
wea nombre = "Claudia Herrera"
wea edad = 38
wea casada = verdad
```

Una `wea` puede cambiar de valor en cualquier momento:

```wn
wea presidente = "Pinochet"
presidente = "Aylwin"
```

Y también puede cambiar de tipo:

```wn
wea estado = "soltero"
estado = 42
estado = verdad
```

<Callout type="warning">
  El tipado dinámico es conveniente, pero con poder viene responsabilidad. Si después haces `estado + 10` y `estado` resulta ser `"casado"`, WN++ te va a retar.
</Callout>

## `duro` - constante inmutable

Usa `duro` cuando el valor no debe cambiar nunca.

```wn
duro TEMPORADAS_LOS_80 = 7
duro PI = 3.1415
duro GRITO_DE_CONDORITO = "¡Plop!"
```

Intentar modificar una constante es un error en ejecución:

```wn
duro campeon = "Colo-Colo"
campeon = "La U"
```

<Callout>
  Convención: las constantes suelen escribirse en `MAYUSCULAS_CON_GUION`. No es obligatorio, pero ayuda.
</Callout>

## Scope de bloque

Las variables en WN++ tienen **alcance de bloque**: solo existen dentro del `{}` donde fueron declaradas.

```wn
wea sospechoso = "Pancho"

cachai (sospechoso == "Pancho") {
  wea coartada = "estaba en el mall"
  lorea(coartada)
}

lorea(coartada)
```

Pero una `wea` declarada afuera si es visible desde adentro, y se puede modificar:

```wn
wea puntaje = 0

para (episodio en [1, 2, 3, 4, 5]) {
  puntaje = puntaje + 10
}

lorea(puntaje)
```

## Referencia rapida

| Palabra | Mutable | Alcance | Cuando usarla |
| --- | --- | --- | --- |
| `wea` | Si | bloque | Casi siempre |
| `duro` | No | bloque | Valores que nunca deben cambiar |

## Errores comunes

<Tabs items={['Usar antes de declarar', 'Olvidar la wea', 'Modificar una constante']}>
  <Tab value="Usar antes de declarar">

  ```wn
  lorea(personaje)
  wea personaje = "Condorito"
  ```

  WN++ lee de arriba hacia abajo. No hay elevación automática de declaraciones.

  </Tab>
  <Tab value="Olvidar la wea">

  ```wn
  nombre = "Yayita"
  ```

  Toda variable debe declararse con `wea` o `duro` antes de usarse.

  </Tab>
  <Tab value="Modificar una constante">

  ```wn
  duro VILLANO = "Pepe Cortisona"
  VILLANO = "Washington"
  ```

  Si necesitas que cambie, usa `wea`.

  </Tab>
</Tabs>
